Map Project
User choose the year and coordinates which he wants to explore.
Program returns map with 10 nearest locations where movies were filmed.
User also can see on the map marked locations of capitals of all EU members.

Prerequisites
	Modules you need to install: 
		1) folium – pip3 install  folium
		2) geopy – pip3 install geopy
Usage:
	1) Enter year
	2) Enter latitude
	3) Enter longtitude
	4) open writed HTML file
Information:
	This program return HTML file, blue markers – nearest locations where the films were made, red markers – capitals of all EU members.
